Detailed Description

Returns the reverse of another function.

For example, a linear up ramp will become a linear down ramp: 

   | Before:      | After:       |
   |           ## | ##           |
   |         ##   |   ##         |
   |       ##     |     ##       |
   |     ##       |       ##     |
   |   ##         |         ##   |
   | ##           |           ## |
